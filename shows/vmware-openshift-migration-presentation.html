Microsegmentation: Tigera Calico Enterprise vs VMware NSX-T

Cloud-native security with zero-trust networking for OpenShift

VMware NSX-T Architecture

Key Characteristics

  • • Hypervisor-dependent (ESXi)
  • • Centralized control plane
  • • Distributed firewall in kernel
  • • VLAN/VXLAN overlay networks
  • • VM-centric policies

Tigera Calico Enterprise Architecture

Key Characteristics

  • • Cloud-native (Kubernetes)
  • • Distributed control plane
  • • eBPF for high performance
  • • Direct routing (no overlay)
  • • Unified Pod/VM policies

Microsegmentation Feature Comparison

Feature VMware NSX-T Tigera Calico Enterprise
Policy Model VM Groups & Security Tags Kubernetes Labels & Selectors
Workload Types VMs Only VMs + Containers + Pods
Performance Kernel Module eBPF (Faster)
Integration vCenter Dependent Kubernetes Native
Compliance Basic Reporting PCI, SOC2, HIPAA Reports
Threat Detection IDS/IPS Add-on Built-in WAF, IDS/IPS
Encryption IPSec Tunnels WireGuard (Faster)

{
